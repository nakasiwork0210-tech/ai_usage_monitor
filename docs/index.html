Usage Monitor

Claude Code / Codex (ChatGPT) の利用量を
Windows タスクトレイに常駐表示

macOS の CodexBar の Windows 版。ログイン不要・完全オンデバイス。

⬇ Windows 版をダウンロード UsageMonitor.exe
無料・オープンソース (MIT) ・ Python 不要 ・ ソースコード
Windows 10 / 11 対応 ・ 約 25 MB

できること

📊トレイに使用率

Claude / Codex の使用率をバーで常時表示。緑・黄・赤で残量がひと目でわかる。

⏱リセットまで表示

5時間・週の各枠が何時にリセットされるか、残り時間をカウントダウン。

📈コストダッシュボード

ローカルログから日次コストを集計。Today / 7日 / 30日で切り替え。

🌗ダーク / ライト

Windows の外観設定に自動追従。手動固定も可能。

🔔使いすぎ通知

使用率が 90% を超えると Windows 通知でお知らせ。

🔒安全設計

独自ログインなし・パスワード保存なし・クラウド同期なし。

使い方

  1. UsageMonitor.exe をダウンロードして実行(Python 不要)
  2. タスクトレイに常駐 — 左クリックでダッシュボード、右クリックでメニュー
  3. Claude Code / Codex CLI でログイン済みなら、そのまま利用量が表示されます
署名なしの exe のため、初回起動時に Windows SmartScreen が警告することがあります。 その場合は [詳細情報] → [実行] で起動できます。中身が気になる場合は ソースからビルドしてください。

セキュリティ

ローカルデータのみ

Claude / Codex CLI が保存済みの認証情報を読むだけ。独自のログイン画面はありません。

パスワード保存なし

トークンを別の場所へコピーしません。更新時も各 CLI の元ファイルに書き戻すだけ。

クラウド同期なし

テレメトリや外部サーバーへの送信は一切ありません。

公式 API のみ

通信は利用量取得のための OpenAI / Anthropic 公式エンドポイントへの HTTPS のみ。コスト集計は完全オフライン。
